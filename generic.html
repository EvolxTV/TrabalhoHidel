Diferenças entre Git e Subversion (SVN)

Git - É um controlador de versão bastante diferente do Subversion, pois seu modelo de versionamento é descentralizado (não há um fluxo central, e quando ele existe, não deve ser modificado, recebendo apenas merges de outros fluxos de desenvolvimento) e o envio de arquivos é realizado em duas fases.

Subversion (SVN) - É um sistema controlador de versão que utiliza um modelo de versionamento centralizado comumente empregado em projetos Java. É a evolução do modelo do CVS, com a introdução dos comandos Rename e Move, que não apenas renomeia/move o arquivo como mantém seu histórico de alterações.
